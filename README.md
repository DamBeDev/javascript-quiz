# Midudev class project to work with Zustand

## Stack

1. Vite + Bun

2. React

3. TypeScript + SWC

4. Zustand


## Instructions

1. Clone de project.

2. Install the dependencies:
```bash
bun install
```

3. Compile the project:
```bash
bun run build
```

4. Run the project on dev mode:
```bash
bun dev
```
